export 环境变量
Use `export` for environment variables:
    HOME, PATH, LD_LIBRARY_PATH, EDITOR, VISUAL, DISPLAY, XAUTHORITY, LESS, http_proxy

set Shell 变量
Do NOT use export for shell variables:
    PS1, IFS, HISTFILE
